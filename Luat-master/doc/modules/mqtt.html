Functions

client (clientId, keepAlive, username, password, cleanSession, will)
mqtt.client() 创建mqtt client实例

Parameters:

  • clientId
  • keepAlive 心跳间隔，默认300秒
  • username 用户名为空请输入nil
  • password 密码为空请输入nil
  • cleanSession 1/0
  • will 遗嘱参数(Last Will/Testament), will.flag, will.qos, will.retain, will.topic, will.payload

Returns:

    result true - 成功，false - 失败

Usage:

    mqttc = mqtt.client("clientid-123", nil, nil, false)
mqttc:connect (host, port, transport)
连接mqtt服务器

Parameters:

  • host 地址
  • port 端口
  • transport tcp

Returns:

    result true - 成功，false - 失败

Usage:

    mqttc = mqtt.client("clientid-123", nil, nil, false); mqttc:connect("mqttserver.com", 1883, "tcp")
mqttc:subscribe (topic, qos)
mqtt.client:subscribe(topic, qos)

Parameters:

  • topic
  • qos 0/1/2

Returns:

    result true - 成功，false - 失败

Usage:

    mqttc:subscribe("/abc", 0) -- subscribe topic "/abc" with qos = 0
mqttc:subscribe({["/topic1"] = 0, ["/topic2"] = 1, ["/topic3"] = 2}) -- subscribe multi topic
mqttc:publish (topic, payload, qos, retain)
mqtt.client:publish(topic, payload, qos, retain)

Parameters:

  • topic
  • payload
  • qos 0/1/2, default 0
  • retain

Returns:

    mid mqtt message id

Usage:

    mqttc = mqtt.client("clientid-123", nil, nil, false)
mqttc:connect("mqttserver.com", 1883, "tcp")
mqttc:publish("/topic", "publish from luat mqtt client", 0)
mqttc:receive (timeout)
mqtt.client:receive([timeout])

Parameters:

  • timeout

Returns:

  1. result true - 成功，false - 失败
  2. data 如果成功返回的时候接收到的服务器发过来的包，如果失败返回的是错误信息

Usage:

    true, packet = mqttc:receive()
false, error_message = mqttc:receive()
