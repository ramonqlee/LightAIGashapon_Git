setup (pin, val)
自适应GPIO模式

Parameters:

  • pin ，参数为pio.P01-31 和 pioP1_1-31 (IO >= 32 and IO - 31)
  • val ，输出模式默认电平：0 是低电平1是高电平，中断模式为回调函数

Returns:

  1. function ,返回一个函数，该函数接受一个参数用来设置IO的电平
  2. nil,中断模式捆绑回调函数返回空

Usage:

  • pins.setup(pio.P1_1,0,key_int) ，配置Key的IO为pio.32,中断模式,自动触发key_int(msg)函数
  • led = pins.setup(pio.P1_1,0) ,配置LED脚的IO为pio.32，输出模式，默认输出低电平。led(1)即可输出高电平
  • key = pins.setup(pio.P1_1),配置key的IO为pio.32，输入模式,用key()即可获得当前电平
